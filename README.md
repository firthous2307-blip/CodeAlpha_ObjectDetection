# Object Detection and Tracking

## Description

This project detects and tracks objects in a video using YOLOv8 and OpenCV. The system identifies objects such as people, vehicles, and other common items and displays bounding boxes around them in real time.

## Features

* Real-time object detection
* Object tracking across video frames
* Detection of people, motorcycles, cars, and other objects
* Bounding boxes and labels displayed on the video

## Technologies Used

* Python
* OpenCV
* YOLOv8 (Ultralytics)

## How to Run

1. Install the required libraries:

pip install -r requirements.txt

2. Place a video file named video.mp4 in the project folder.

3. Run the program:

python app.py
